# Examples

Stately ships packaged examples under `src/lib/examples/` so consumers can inspect patterns that match the public API exactly.

## Included examples

- option stores
- setup stores
- persistence
- history and time travel
- sync
- async orchestration

## How to use them

Treat the example files as reference implementations for your own stores.
They are especially useful when you want a typed example for plugin configuration without reading the runtime internals.
